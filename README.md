## Black Thursday

Find the [project spec here](https://github.com/turingschool/curriculum/blob/master/source/projects/black_thursday.markdown).

This project is a system that is able to load, parse, search and execute business information.

Data is loaded through CSV files and includes information about merchants, items, invoices, invoice items, transactions, and customers.

Information is parsed and creates connections so that one can analyze data through the Sales Analyst class.

Some of the functionality of Sales Analyst includes but is not limited to, identifying popular items, finding a merchant's average item price, finding a merchant's average invoice count, finding total revenues by date, finding merchant by item id, and finding merchants with only one item.


## Installation

In order to see this project work, run tests using 'Rake test' on your command line or 'mrspec'. If you would like to run mrspec, gem install mrspec.
